# 灵兽迷踪性能修复包

本包将网页运行时图片从约 52.8 MB 压缩到约 1.14 MB，适合手机网络加载。

## 上传步骤

1. 在 GitHub 仓库根目录上传并覆盖 `index.html`、`app.js`、`styles.css`、`assets-check.html`。
2. 进入仓库的 `assets` 文件夹，上传本包 `assets` 目录中的全部 30 张 `.jpg` 图片。
3. 图片名称相同的文件选择覆盖。旧的 `.png` 图片不必删除，新代码不会再请求它们。
4. 等待 GitHub Pages 部署后，使用 `Ctrl + F5` 强制刷新网页。

访问 `/assets-check.html`。显示“全部 30 张图片已成功加载”即可。
